Car Rent App

Приложение для просмотра и аренды автомобилей с возможностью добавления в избранное.

Stack: Kotlin, Jetpack Compose, MVVM, Firebase (Auth, Firestore), Coroutines  

Архитектура: 
UI → ViewModel → Repository → Firebase  

- Работа с Firestore (хранение автомобилей и избранного)  
- Авторизация пользователей (Firebase Auth)  
- Управление состоянием через Compose  
- Отображение списков и деталей автомобилей
